# 精品咖啡豆订购网站

一个功能完整的咖啡豆在线订购和管理系统，采用现代化的响应式设计，以棕色为主题色调。

## 功能特性

### 用户功能
1. **产品浏览**
   - 查看所有咖啡豆产品
   - 产品搜索功能
   - 按产地和烘焙度筛选
   - 多种排序方式（价格、名称、评分）

2. **购物车**
   - 添加/移除商品
   - 修改商品数量
   - 实时计算总价

3. **用户账户**
   - 用户注册和登录
   - 个人资料管理
   - 订单历史查看
   - 收藏夹功能

4. **订单系统**
   - 在线下单
   - 订单状态跟踪（待处理、处理中、已发货、已送达）
   - 订单详情查看

5. **个性化推荐**
   - 基于购买历史的智能推荐

### 管理员功能
1. **产品管理**
   - 添加新产品
   - 编辑产品信息
   - 删除产品
   - 库存管理

2. **订单管理**
   - 查看所有订单
   - 更新订单状态
   - 订单详情查看

3. **数据统计**
   - 总销售额
   - 订单总数
   - 产品总数
   - 用户总数

## 使用方法

### 默认管理员账户
- 邮箱：`admin@coffee.com`
- 密码：`admin123`

### 启动网站
1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器（推荐）：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js (需要安装 http-server)
   npx http-server
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 技术栈
- HTML5
- CSS3 (响应式设计)
- JavaScript (原生ES6+)
- LocalStorage (数据持久化)
- Font Awesome (图标库)

## 数据存储
所有数据（产品、用户、订单等）都存储在浏览器的 LocalStorage 中，刷新页面后数据不会丢失。

## 浏览器兼容性
支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）

## 设计特点
- 棕色主题，符合咖啡文化
- 响应式设计，支持移动端和桌面端
- 现代化的UI/UX设计
- 流畅的交互动画
